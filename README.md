## ğŸ“– Overview
**AgentForge** is a high-performance, low-code platform built for the **Winter Solstice Hackathon 2026**. It allows users to orchestrate complex AI behaviors through a visual node-based system. Our platform doesn't just call an API; it manages a resilient ecosystem of autonomous agents designed for scale and reliability.

## âš¡ Technical Highlights (The "Judge's Choice")
* **Zero-Downtime Retries:** Our custom `botService` implements sophisticated "Back-off & Retry" logic. When Gemini hits a 429 Rate Limit, AgentForge calculates the required delay and self-heals without crashing the UI.
* **Dynamic Model Discovery:** We query the Google AI models list at runtime to always utilize the latest "Flash" model available in the user's region, ensuring future-proof compatibility.
* **Massive Scale Processing:** Architected to handle high-density package logic, currently managing over **2,318,229+ lines of code** across its integrated dependencies.
* **Atomic State UI:** A fully responsive, scroll-locked chat interface that remains performant even with deep message histories and heavy token usage.
